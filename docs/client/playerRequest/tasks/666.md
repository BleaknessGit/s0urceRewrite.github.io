## Name

Restart Computer Task

## Description

Used to restart the Computer<br>
Sent after Restart button is pressed

## Raw Packet

`42["playerRequest",{"task":666"}]`

## JSON

``` json
[
    "playerRequest",
    {
        "task": 666,
    }
]
```

## JSON Definition

##### Main Object
| Key  | Type   | Example | Description |
|------|--------|---------|-------------|
| task | Number | 666     | Task ID     |
